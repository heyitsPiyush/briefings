This month, Microsoft's June 2026 Patch Tuesday addressed a significant number of vulnerabilities, including six zero-days, five of which were publicly disclosed and one actively exploited. A critical remote code execution (RCE) flaw, CVE-2026-45657, was identified in the Windows Kernel, stemming from use-after-free conditions in TCP/IP data processing, allowing unauthenticated remote attackers to execute code with SYSTEM privileges. Another notable RCE, CVE-2026-47291, affected HTTP.sys due to integer overflow and heap-based buffer overflow flaws, also enabling unauthenticated remote code execution with low attack complexity, though systems with default MaxRequestBytes registry values may be unaffected. Additionally, CVE-2026-44815, a critical RCE in Windows DHCP Client Service, was due to a stack-based buffer overflow that could be exploited by a rogue DHCP server to achieve code execution.

Beyond Microsoft, a proof-of-concept (PoC) exploit for CVE-2026-49975, dubbed the "HTTP/2 Bomb," was released, targeting Apache HTTP Server versions 2.4.17 through 2.4.67, allowing remote denial-of-service via memory exhaustion. Adobe also patched CVE-2026-48291, a heap-based buffer overflow in USD-Fileformat-plugins leading to remote code execution. In the Linux kernel, CVE-2026-46331, a net/sched packet-editing bug, was fixed, which could lead to page cache corruption via partial copy-on-write handling, a type of memory corruption that often serves as a primitive for local privilege escalation. Microsoft Defender also saw a publicly disclosed elevation of privilege zero-day, CVE-2026-50656, named "RoguePlanet," which exploits a race condition to grant SYSTEM-level privileges, with a public PoC available that bypasses real-time protection.
