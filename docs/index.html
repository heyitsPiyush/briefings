Hardware Hacking

deep

The Android June 2026 security update addressed 124 vulnerabilities across Android 14, 15, and 16, including a privilege-escalation flaw (CVE-2025-48595) in the Framework component that is actively being exploited in targeted attacks. Another significant firmware vulnerability was found in GL.iNet GL-MT3000 routers up to version 4.4.5, allowing remote command injection via the LuCI JSON-RPC Interface (CVE-2026-11447) and MiniDLNA service (CVE-2026-11449), as well as within the OpenVPN Client Import Workflow (CVE-2026-11448). NIST also released an initial public draft on Hardware-Enabled Security: Confidential Computing of Data in Cloud Workloads, focusing on protecting data in use, particularly for AI workloads on cloud infrastructures. Furthermore, Cisco disclosed an authenticated privilege escalation vulnerability (CVE-2026-20245) in its Catalyst SD-WAN Manager, Controller, and Validator, which allows local attackers with netadmin privileges to execute arbitrary commands as root.

IoT security remains a critical area with a significant increase in botnet activity, as malicious IoT botnets are increasingly used for DDoS attacks, with one, Aisuru, achieving a record 29.7 Tbps attack in Q3 2025. Network equipment, including routers and IP cameras, has become a primary target, with a 124% jump in IoT malware attacks in 2024, many against government facilities. The vulnerability in iRM-IEI Remote Management solution, designed for embedded systems and IoT devices, exposes system configuration to unauthenticated remote adversaries due to missing authentication. In supply chain security, a leaked Miasma credential-stealing attack framework, dubbed Hades, targets public package registries like PyPI and npm, GitHub repositories, and CI/CD pipelines by exploiting compromised developer accounts and injecting malicious code into open-source packages. New hardware hacking tools like Flipper Zero, USB Rubber Ducky, Proxmark3, and HackRF One continue to be essential for ethical hackers in wireless and RF signal research.
