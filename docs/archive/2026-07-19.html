Latest exploit development news as of July 19, 2026, includes several critical vulnerabilities and public proof-of-concept (PoC) releases across various platforms. A significant Linux kernel local privilege escalation flaw, GhostLock (CVE-2026-43499), a use-after-free in the real-time mutex priority-inheritance code, was disclosed and patched for AlmaLinux versions 8, 9, and 10, allowing unprivileged local users to gain root and escape containers. Another critical Linux kernel vulnerability, Januscape (CVE-2026-53359), affecting nested virtualization in KVM on Intel and AMD x86_64 systems, also had a public PoC demonstrating hypervisor host crashes from a guest. Additionally, a Windows zero-day PoC named LegacyHive, targeting the User Profile Service for arbitrary hive loading and privilege escalation, was released by researcher Chaotic Eclipse, functioning on fully patched Windows desktop and server versions from July 2026 Patch Tuesday.

Browser security updates are prominent, with Google Chrome version 150.0.7871.128/.129 fixing CVE-2026-15905, a high-severity use-after-free in Chromium's Aura UI framework. Firefox 152 also addressed over 40 vulnerabilities, including critical use-after-free, sandbox escape, and privilege escalation bugs like CVE-2026-12289 and CVE-2026-12291. Microsoft's July 2026 Patch Tuesday addressed a record 622 vulnerabilities, including two actively exploited zero-days, CVE-2026-56155 in Active Directory Federation Services and CVE-2026-56164 in Microsoft SharePoint Server, both enabling elevation of privilege. Memory corruption vulnerabilities were also addressed, such as CVE-2026-44747, a critical memory corruption flaw in SAP NetWeaver Application Server ABAP with a CVSS score of 9.9, allowing authenticated attackers to access or modify data, or cause system unavailability. Public exploits were also released for "wp2shell" remote code execution flaws (CVE-2026-63030 and CVE-2026-60137) affecting WordPress Core versions 6.9.
