Exploit Development

deep

Exploit development in May 2026 has seen a significant focus on Linux kernel vulnerabilities and browser exploits, with several high-impact disclosures. Notably, the Linux kernel was affected by "Copy-Fail" (CVE-2026-31431), a local privilege escalation flaw with a small Python PoC capable of gaining root on major distributions since 2017. This was followed by "Dirty Frag" (CVE-2026-43284 and CVE-2026-43500), another Linux kernel LPE chain without a race condition, reliably affecting various Linux distributions. Furthermore, "DirtyDecrypt" (CVE-2026-31635), a variant of the Copy-Fail family, also saw a public PoC release and exploits a missing copy-on-write (COW) guard in the rxgkdecryptskb function, impacting distributions with CONFIG_RXGK enabled.

Browser security also featured prominently, with Google Chrome receiving updates for critical vulnerabilities including CVE-2026-9111, a use-after-free in WebRTC allowing remote code execution on Linux via crafted HTML, and CVE-2026-9110 for UI spoofing on Windows after renderer compromise. Additionally, a long-unpatched Chromium bug related to Service Workers and the Background Fetch API has been publicly detailed, allowing persistent JavaScript execution across browser restarts. Microsoft Edge also addressed multiple vulnerabilities, including CVE-2026-9110 and CVE-2026-9111, indicating shared Chromium codebase issues. Beyond this, memory corruption research showed an AI-assisted macOS kernel exploit on Apple M5 hardware running macOS 26.4.1, demonstrating AI's growing role in vulnerability discovery.
