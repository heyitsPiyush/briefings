Bug Bounty

deep

The bug bounty landscape in May 2026 has been significantly influenced by the increasing role of AI in vulnerability discovery. Google has restructured its Vulnerability Reward Programs (VRPs) for Android and Chrome, boosting Android bounties to a maximum of $1.5 million for high-severity exploits, particularly those targeting Pixel's Titan M security chip. Conversely, Chrome rewards have seen reductions, a change attributed to AI-assisted fuzzing and static analysis tools lowering the barrier for finding certain browser vulnerabilities. This shift incentivizes researchers to focus on harder-to-find issues and attack surfaces less covered by automated tools. Additionally, Anthropic's Claude Mythos Preview, an AI model, has reportedly identified over 1,500 confirmed critical vulnerabilities across open-source projects, with maintainers in some instances requesting a slowdown due to the volume of reports.

Microsoft's May 2026 Patch Tuesday addressed 130 CVEs, including 30 critical vulnerabilities. Notable disclosures include CVE-2026-42826, a critical information disclosure in Azure DevOps with a CVSS score of 10, and CVE-2026-33109 and CVE-2026-33844, critical RCE flaws in Azure Managed Instance for Apache Cassandra with CVSS scores of 9.9 and 9.0 respectively. Another significant disclosure involved a critical SQL injection vulnerability, CVE-2026-26980 (CVSS 9.4), in Ghost CMS, which was exploited to hijack over 700 websites through malicious JavaScript injections. Bugcrowd's platform updates in May include improved identity verification for researchers, added IP restrictions for customers, and a new API version, while HackerOne shows active programs with varying bounties and recent updates to programs like GoCardless and Adobe. Intigriti also released product updates, including "Intigriti Insights" for security teams and "Triage Assist" to improve crowdsourced security.

Hardware Hacking

deep

The embedded security landscape in May 2026 highlights several critical vulnerabilities and evolving attack methodologies. A significant side-channel attack named FROST has emerged, allowing malicious websites to track user activity by exploiting SSD timing patterns through the Origin Private File System (OPFS) API on macOS (Apple M2) and Linux (AMD Ryzen 7 5800X3D) systems. This technique can exfiltrate data from native applications to a malicious website and enable website fingerprinting with high accuracy. Furthermore, firmware vulnerabilities remain a persistent threat, with a command injection flaw (CVE-2026-38707) discovered in InHand Networks IR302, IR305, IR315, and IR615 series industrial routers, potentially granting remote ROOT privileges to attackers. Additionally, the RondoDox botnet is actively exploiting older vulnerabilities like CVE-2018-5999 in ASUS AsusWRT routers, impacting devices that haven't received recent firmware updates.

IoT security research reveals a shift towards automated AI tools for exploiting unmanaged devices and for lateral movement within corporate networks, with one in three data breaches now involving an IoT device. A critical WolfSSL vulnerability (CVE-2026-5194) with a CVSS score of 9.1 has been identified, affecting the embedded TLS library used in automotive systems, industrial controllers, and various IoT devices. Supply chain security also faces growing challenges, with a breach at Foxconn’s North American manufacturing operations by the Nitrogen ransomware group resulting in the exfiltration of confidential project documentation and network topology maps belonging to major tech companies like Apple, Google, Nvidia, Dell, and Intel. New hardware hacking tools like the Flipper Zero continue to be prominent for wireless protocol analysis and RFID/NFC interactions, illustrating the ongoing need for robust physical and digital security measures.

Exploit Development

deep

Exploit development in May 2026 saw significant activity across various platforms, particularly in Linux kernels and web browsers. A critical Linux kernel vulnerability, CVE-2026-46333, disclosed by Qualys, allows an unprivileged local user to escalate privileges to root, affecting popular distributions like Debian 13, Ubuntu 24.04 and 26.04, and Fedora 43 and 44. This logic flaw, residing in _ptracemayaccess(), enables attackers to read sensitive files like /etc/shadow or steal SSH host private keys. Another notable Linux kernel flaw, CVE-2026-43305, a privilege escalation vulnerability in the AMD GPU display driver, affects local availability by causing a kernel hang on systems without FAMS support, tracked under CWE-667 (Improper Locking). Additionally, May saw over 1756 new Linux vulnerabilities addressed, including a critical heap-based buffer overflow in NGINX's ngxhttprewritemodule (CVE-2026-42945) with an associated public PoC for the "DirtyDecrypt" flaw.

Browser exploits were also prominent, with Google Chrome receiving updates for multiple high-severity vulnerabilities, including CVE-2026-9111, a use-after-free in WebRTC allowing remote code execution on Linux via crafted HTML pages. This highlights memory corruption as a continuing attack primitive. Furthermore, Microsoft released mitigations for "YellowKey" (CVE-2026-45585), a Windows zero-day BitLocker bypass that allows physical attackers to access encrypted data by abusing the Windows Recovery Environment (WinRE) through specially crafted 'FsTx' files on a USB drive. The Zero Day Initiative (ZDI) published numerous advisories, and Pwn2Own Berlin 2026 concluded, showcasing various exploits against Windows, Linux, and Safari, with a notable emphasis on AI-assisted vulnerability research. New research also introduced FROST, a browser-side channel attack exploiting the Origin Private File System (OPFS) to infer user activity by analyzing SSD behavior.
