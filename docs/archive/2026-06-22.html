Recent hardware hacking and embedded security news highlights significant vulnerabilities and new research. Paradigm Shift researchers recently unveiled "usbliter8," an unpatchable BootROM exploit impacting Apple's A12 and A13 chips, including devices like the iPhone XS, XR, iPhone 11 series, and Apple Watch Series 4 and 5. This hardware flaw, rooted in the Synopsys DWC2 USB controller, allows arbitrary code execution within the SecureROM by chaining a USB controller bug with a firmware configuration weakness, requiring physical access to a device in DFU mode with a dedicated RP2350-based microcontroller board. Additionally, a critical firmware vulnerability (CVE-2026-35075) was identified, involving hard-coded credentials embedded in firmware images, which allows unauthenticated remote attackers to gain full administrative access to affected devices. Another command injection vulnerability (CVE-2026-12808) was found in Edimax BR-6478AC V2 1.23, enabling remote arbitrary code execution through the router's web administration interface due to inadequate input validation in the stainfo function.

Similarly, side-channel attacks leveraging Intel's Data Streaming Accelerator (DSA) on Xeon processors, dubbed "DSASSASSIN," have demonstrated cross-VM covert and side-channel capabilities by exploiting vulnerabilities in the Device TLB and shared work queue. The prevalence of IoT security threats continues to grow, with projections exceeding 25 billion connected devices by 2026, leading to increased concerns about botnet recruitment, supply chain compromise, and the exploitation of default credentials and firmware vulnerabilities in devices that often cannot run security agents or receive frequent updates.
