As of July 17, 2026, several critical CVEs are under active exploitation, prompting urgent action. CISA recently added multiple vulnerabilities to its Known Exploited Vulnerabilities (KEV) Catalog. These include CVE-2026-58644, a critical deserialization flaw (CVSS score 9.8) affecting Microsoft SharePoint Server that allows unauthorized arbitrary code execution, requiring patches by July 19. Additionally, CVE-2026-39808 and CVE-2026-25089, both critical command injection vulnerabilities in Fortinet FortiSandbox, are being actively exploited, allowing unauthenticated remote code execution. CISA also highlighted CVE-2026-56155 (CVSS 7.8), an elevation of privilege in Microsoft Active Directory Federation Services, and CVE-2026-56164 (CVSS 5.3), a missing authentication flaw in SharePoint Server, both confirmed to be exploited in the wild. Earlier in July, CISA added CVE-2026-48282, an Adobe ColdFusion path traversal vulnerability, to the KEV catalog.

Active threat intelligence reveals a dynamic landscape of ransomware and malware campaigns. The Gentlemen ransomware group has been highly active, with ransomware incidents increasing by 33% from June 2025. A novel, autonomous ransomware operation named JadePuffer has emerged, exploiting CVE-2025-3248 in Langflow instances to conduct intrusions, exfiltrate data, and issue extortion demands using large language models. Furthermore, a global phishing campaign is distributing stealthy malware such as Agent Tesla, Remcos, XWorm, and a Best Private LOGGER variant through fake TrueType Font (TTF) files. Microsoft Defender Experts also observed increased activity from ACR Stealer, which uses ClickFix lures to steal browser credentials and sensitive documents from enterprise environments. The group UAT-7810, linked to China, continues to compromise internet-facing networking devices, exploiting unpatched Ruckus and ASUS devices to expand their operational relay box infrastructure.
